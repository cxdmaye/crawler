## 使用方法

### 快速构建 Windows exe

```bash
# 使用 Makefile (推荐)
make build-windows

# 使用构建脚本
./build-windows.sh

# 直接使用 wails 命令
wails build -platform windows/amd64 -clean -o crawler-app.exe
```

### Windows 系统上构建

```powershell
# 使用 PowerShell 脚本
.\build-windows.ps1

# 带清理选项
.\build-windows.ps1 -Clean

# 调试模式
.\build-windows.ps1 -Debug
```
